# Numpy Array Operations 
## Suma
Primero definimos 2 variables, en este caso son 2 listas 
```python

arr1 = [4,5,6]
arr2 = [7,8,9]
```
```python

import numpy as np

arr1 = np.array([4, 5, 6])
arr2 = np.array([7, 8, 9])
resultado = arr1 + arr2
print(resultado)  # array([11, 13, 15])
```
